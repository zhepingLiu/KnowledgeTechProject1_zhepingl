The file WordCorrection_main.py is written in Python 3.7.
Used external library editdistance.
Before running the program, editdistance needs to be installed.
    - It can be installed using command
        "pip install editdistance" in Terminal
To run the program, type
    "python3 WordCorrection_main.py" in Terminal

NOTE: due to the complexity of the computation, the time spent in this program
can be very long.
